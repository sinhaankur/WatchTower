What This Project Uses From Upstream

This section keeps the provenance story explicit while matching the application’s panel-based information layout.

Upstream Dependencies

  • Container updater behavior: Uses upstream containrrr/watchtower for image polling and restart semantics.
  • Networking concept direction: UX and control flow are inspired by tailscale/tailscale for team-friendly host connectivity workflows.
  • Domain connectivity: Cloudflare tunnel setup is generated through WatchTower Host Connect helper endpoints.
