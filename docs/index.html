Safe Website Checklist

  • Protect deploy endpoints with WATCHTOWER_TRIGGER_TOKEN
  • Use key-only SSH on nodes, disable password auth
  • Use limited sudoers commands for service reload
  • Run Trivy checks on pull requests before merge
  • Keep rollback-capable backups for each app

Version-Controlled Release Flow

  1. Bump watchtower.__version__ in watchtower/__init__.py
  2. Merge changes to main
  3. Create semantic tag like v1.2.1
  4. GitHub Actions auto-publishes to GHCR, PyPI, and GitHub Releases
./scripts/release.sh 1.2.1

Install and Use Across Compatible Ecosystems

  • Python ecosystem (pip): pip install watchtower-podman
  • Container ecosystem (OCI): ghcr.io/sinhaankur/watchtower:latest
  • Linux service ecosystem: use systemd/watchtower.service
  • Windows/macOS script ecosystem: use install_windows.ps1 and install_macos.sh
