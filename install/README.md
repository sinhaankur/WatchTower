# Install Scripts

Platform-specific one-shot installers for WatchTower.

| Script | Platform | Description |
|--------|----------|-------------|
| `install.sh` | Linux / macOS | Universal installer (auto-detects OS) |
| `install_app_center.sh` | Linux | Full App Center service install (`/etc/watchtower`) |
| `install_macos.sh` | macOS | macOS launchd service install |
| `install_watchtower.sh` | Linux | Minimal daemon install |
| `install_windows.ps1` | Windows | PowerShell installer |
| `install_watchtower_windows.cmd` | Windows | CMD wrapper |
| `run_app_center_macos.sh` | macOS | Launch App Center manually on macOS |
| `run_app_center_windows.ps1` | Windows | Launch App Center manually on Windows |

See the [main README](../README.md) for usage instructions.
